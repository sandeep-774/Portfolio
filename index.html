UI Design

User interface (UI) design is likely the first thing you encounter when you use an application or visit a website. User interface design is responsible for a product's appearance, interactivity, usability, behaviour, and overall feel.

Frontend Developer

A Front-End Developer is someone eho creates websites and web application. The differnce between Front-End and Back-End is that Front-End refers to how a web page looks, while Back-End refers to how it works.

Backend Developer

Back-end developer are the experts who build and maintains the mechanism that process data and perform action on websites. Unlike front-end developers, who control everything you can see on a website, backend developers are involved in data storage, security, and other side function that you cannot see.
